# Luthando Mbuyane Portfolio

Static portfolio website for graduate and junior software engineering roles.

## Focus

This portfolio positions Luthando as a junior backend/full-stack developer with:

- Computer Science and Psychology background from UCT
- Python, Java and SQL experience
- FastAPI, Flask, PostgreSQL, MongoDB and SQLite projects
- REST APIs, authentication, testing, CI/CD and deployment exposure
- HCI, accessibility and people-centred design perspective

## Run locally

Open `index.html` in a browser.

For a simple local server:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## Deploy

This is a static site and can be deployed on GitHub Pages, Netlify, Vercel or any static host.

## Notes

The repository includes placeholders for images, diagrams and the downloadable CV. Replace placeholder files with actual assets before deploying.
