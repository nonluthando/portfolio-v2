ScoreRent

Backend · Live

ScoreRent is a live, iteratively developed backend web application that helps renters evaluate property listings before paying application fees.

Overview

Renters often pay non-refundable application fees with limited information about whether a listing is worth pursuing. ScoreRent provides a structured, criteria-based way to evaluate listings, helping users make informed decisions before committing money.

My Role

I designed and implemented the backend system end-to-end, including API design, scoring logic, input validation, and deployment. The system was built to support ongoing iteration rather than a one-off implementation.

Key Features

  • Criteria-based scoring logic
  • Typed input validation
  • RESTful API structure
  • Live deployment for real-world testing
