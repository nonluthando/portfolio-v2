SeaClear

Human computer interaction project focused on making beach water quality information easier to access, understand, and act on.

HCI Usability Accessibility Flask User-centred design Documentation

Overview

SeaClear is a group-built HCI system designed to support beachgoers by improving the clarity and usability of water quality information. The project combines public-facing communication with interactive features that help users report issues and engage with the platform.

My contributions

My work focused on user-friendly design, improving key site flows, and ensuring features were usable and well documented.

HCI approach

Because SeaClear is an HCI-focused project, I helped keep the user experience consistent by ensuring the team’s design decisions followed usability principles and heuristic guidelines. This included prioritising clear navigation, meaningful feedback, readable information structure, and predictable user flows across pages.

Usability heuristics Consistency Meaningful feedback Clear navigation Information structure

Community reporting feature

I built the user-facing community report flow, including the reporting page and form, adding a beach selection dropdown and connecting the submission to backend handling so reports could be saved correctly.

Reporting flow Form UX Backend integration

Homepage revamp

I improved the home page layout and information structure, including beach card layout changes, map placement adjustments, and carousel improvements to support a smoother entry experience for users.

Layout improvements Map widget Carousel

News section improvements

I restructured the news tab for better readability and navigation, and linked the homepage “More News” to a dedicated page so users could explore full news content without friction.

Information structure Navigation UX Content clarity

Education page contributions

I helped restructure educational content into clearer sections and added or updated content focused on practical impacts, including waterborne illness information and economic impact content. I also supported multimedia additions through video links.

Educational content Multimedia Clarity-first design

Base UI and navigation UX

I improved navigation and base layout patterns across the site by updating the navbar, strengthening UI consistency, improving flash messages, and reducing confusion in key user actions.

Navbar UX Flash messaging UI consistency

Documentation and quality assurance

I contributed to the project report, authored test cases to validate key flows, and wrote a user manual to support usability and reduce onboarding friction. This improved evaluation quality and strengthened maintainability.

Test cases User manual Project report QA thinking

What I learned

SeaClear strengthened my HCI approach to development. I learned how to translate usability thinking into concrete changes across navigation, information structure, and interface flows. It also reinforced how important documentation and structured testing are when building systems meant for public use.

HCI mindset Usability iteration Documentation QA discipline
